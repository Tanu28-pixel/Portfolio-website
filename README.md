# Personal Portfolio Website

## Overview

A responsive personal portfolio website developed using HTML, CSS, and JavaScript to showcase my skills, projects, experience, certifications, and contact information as an aspiring Data Analyst.

## Features

* Responsive design for desktop and mobile devices
* Animated typing effect using JavaScript
* Smooth scrolling navigation
* Interactive project showcase section
* Skills, certifications, and experience sections
* GitHub and LinkedIn integration
* Resume download/view option

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts

## Key Learnings

* Front-end web development fundamentals
* Responsive web design
* CSS animations and transitions
* JavaScript DOM manipulation
* Portfolio design and deployment

## Future Improvements

* Dark/Light mode toggle
* Project filtering functionality
* Contact form integration
* Blog section for analytics projects
